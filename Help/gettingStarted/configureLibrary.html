Step 1: Configuring Library

Kactus2 library settings determine, which folders are scanned for components and other IP-XACT documents.

  1. Click the configure library button on the top ribbon.
  2. Click the plus button.
  3. Select the root folder of the documents as the library path.
  4. Make sure it is both active and default.
  5. Click ok.
  6. If there was any readable IP-XACT documents, they will show up under Library items.

The new documents will be placed under the default location.

The existing documents are scanned from all active locations.

The next step is accessing documents.
