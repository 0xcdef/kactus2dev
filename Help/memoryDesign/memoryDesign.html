Memory Design

Memory design displays the connections made in the associated HW design. These are constructed using the address spaces and memory maps referenced within the connected bus interfaces.

Address spaces are displayed on the left side of the view. Address spaces contain segments.

Memory maps are displayed on the right side of the view. Memory maps contain address blocks, registers and fields.

Overlapping connection areas are displayed in red.

Fields can be extended by using ALT together with the mouse wheel.
