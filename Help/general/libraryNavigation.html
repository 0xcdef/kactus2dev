Library Navigation

How to access IP-XACT documents and open them for editors.

VLNV Tree

Documents are sorted to VLNV tree based on the VLNV tuple of a document.

  1. Select VLNV tree.
  2. Browse to the desired document, double click to open.
  3. Right click for context menu. The additional options include opening and creating designs.
  4. The external interfaces and ad-hoc ports are illustrated in the preview.

Hierarchy Navigation

Documents are sorted to hierarchy display based on their position in designs and sub designs.

  1. Select hierarchy.
  2. Hierarchical components are expandable, displaying the associated designs and the components that have instances in the design.
  3. Hierarchical navigation is recursive, so that a design within a design is also expandable.
  4. This column displays how many instances in the design use the component.

Library Filters

You may limit the number of displayed IP-XACT documents using filters:

  1. Hide/show filters
  2. Only documents which type is checked are displayed. Advanced stands for designs and design configurations.
  3. Hide documents that do not match the implementation. For instance, unchecking HW hides all hardware components and buses.
  4. Hide components that do not match any checked hierarchy level.
  5. Hide components that do not match any checked firmness level.
  6. Hide documents that have no errors or do have errors.
  7. Hide documents that do not have the tags listed here.
